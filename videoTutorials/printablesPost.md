## üñ®Ô∏è Print Settings

### Main Body

- Material: PLA or PETG
- Layer Height: 0.2mm
- Infill: 20%
- Support: None needed
- Build Plate Adhesion: Brim recommended

### Diffuser

- Material: Clear/Translucent PLA/PETG
- Layer Height: 0.2mm
- Infill: 100%
- Support: None
- Special: Pause print after diffuser layers

## üîß Required Materials

### Electronics

- ESP8266 NodeMCU or ESP32 D1 Mini
- WS2812B LED Strip (100 LED/m)
- 5V Power Supply (2A minimum)
- Micro USB cable
- Dupont wires
- 220 Ohm Resistor
- Optional: Power jack connector

### Tools

- Soldering iron & solder
- Wire strippers
- Small screwdriver set
- Hot glue gun
- Basic hand tools
